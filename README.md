## How to Run
1. Clone this repo  
   ```bash
   git clone <your-repo-link>
   cd taskflow-mern-test
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```
4. Make sure MongoDB is running locally (mongodb://127.0.0.1:27017)
